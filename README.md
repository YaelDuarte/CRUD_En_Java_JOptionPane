# CRUD Básico en Java

Aplicación desarrollada en Java utilizando JOptionPane.
Permite agregar, mostrar, buscar y eliminar nombres usando arreglos.

## Tecnologías
- Java SE
- JOptionPane

## Funcionalidades
- Agregar nombres
- Listar nombres
- Buscar nombre
- Eliminar nombre
